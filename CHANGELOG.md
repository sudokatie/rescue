# Changelog

## [0.1.0] - 2026-02-02

Initial release.

### Added

- Core game loop with 60 FPS rendering
- Ship movement with thrust, inertia, and drag
- Horizontal wraparound world (2560px)
- Laser projectile system
- Lander AI (descend, grab humans, rise)
- Human behavior (walk, grabbed, fall, rescue states)
- Collision detection with wraparound support
- Wave system with scaling difficulty
- Radar minimap showing full world
- Score tracking
- Lives system
- Title screen and game over screen
- 171 unit tests
